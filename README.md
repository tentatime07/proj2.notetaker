# proj2.notetaker v1.1

A simple command-line note taker that automatically saves to a JSON file.

Supports:

- Add new notes
- View all saved notes
- Edit existing notes
- Delete individual notes
- Search notes by keyword
- Clear all notes
- Automatically save notes between sessions
- Handle invalid menu selections and empty input

## Future Improvements for v1.2

- Sort notes
- Add UI
